# Valhallan Website Mockup

This is a redesigned mockup for the Valhallan.com website featuring a parent-focused navigation structure and sales-oriented content.

## Features

### New Navigation Structure
The site now focuses on what parents are looking for:
- **LEARN** - Explore the program with a free coaching session
- **PLAY** - Join a team and meet weekly for training
- **TRAIN** - Professional coaching that develops skills and character
- **COMPETE** - Experience pro-level tournaments and leagues

### Design Elements
- **Modern, clean design** using Valhallan brand guidelines
- **Parent-focused messaging** that emphasizes character building, safety, and skills
- **Clear value propositions** inspired by the XP League approach
- **Lead capture** with a “Learn more” form
- **Mobile-responsive** design

### Key Sections

1. **Hero Section**
   - Bold headline: "Where Players Become Legends"
   - Prominent CTAs for free sessions and finding arenas
   - Trust indicators with stats

2. **Value Propositions**
   - Positive Environment
   - Skill Development
   - Pro-Level Experience
   - Community Building

3. **Learn/Explore Section**
   - Free coaching session offer
   - Removes barriers to entry
   - Shows the benefits without commitment

4. **Play Section**
   - Team-based approach
   - Weekly training structure
   - Legendary Path® program

5. **Train Section**
   - Focus on life skills (communication, teamwork, sportsmanship)
   - Professional coaching
   - Character development

6. **Compete Section**
   - Local, national, and championship competition
   - Ragnarök tournament series
   - Virtual and in-person options

7. **Social Proof**
   - Parent testimonials
   - Statistics on player improvement
   - "We Build Character, Not Just Avatars" messaging

8. **Games Section**
   - Showcase popular games offered
   - Clear indication that programs cover what kids already love
